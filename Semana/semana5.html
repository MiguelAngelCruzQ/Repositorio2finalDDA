Semana 5

Primer Proyecto

El sistema CDentis es un proyecto orientado a optimizar la gestión de una clínica dental, permitiendo administrar de forma organizada los datos de pacientes, médicos, citas y expedientes clínicos. Este módulo corresponde a la interfaz de inicio de sesión, donde cada usuario puede ingresar según su rol, garantizando seguridad y un acceso diferenciado entre médicos y recepcionistas. A través de esta plataforma se busca mejorar la atención al paciente, agilizar los procesos administrativos y centralizar la información dentro de una base de datos estructurada. Este prototipo forma parte del desarrollo general del sistema, integrando tanto el diseño visual como la lógica funcional necesaria para futuras expansiones y pruebas en entornos reales.
